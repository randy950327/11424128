# 題目三、熱工程案例：冷卻系統（CPU散熱）

### ▶ 工程問題：電子元件冷卻速度如何預測？

在處理如高階 CPU的散熱設計時，預測溫度隨時間下降的趨勢至關重要。我們可以透過物理建模與微積分運算來達成。

---

### 1. 建模（牛頓冷卻定律）

根據物理原理，物體的冷卻速率與其跟環境的溫差成正比。建立微分方程如下：

$$\frac{dT}{dt} = -k(T - T_{env})$$

- $T$: CPU 當前溫度
- $T_{env}$: 環境溫度（機殼內氣溫）
- $k$: 冷卻常數（代表散熱器的效能）
- $t$: 時間

---

### 2. 求解過程 $T(t) = ?$

我們需要解開的微分方程式為：

$$\frac{dT}{dt} = -k(T - T_{env})$$

這是一個「一階常微分方程式」，我們可以透過**分離變數法 (Separation of Variables)** 來求解。核心概念是將所有的 $T$（溫度變數）移到等號左邊，將所有的 $t$（時間變數）移到等號右邊。

#### **步驟 1：分離變數**

我們將等式兩邊同時乘上 $dt$，並同時除以 $(T - T_{env})$：

$$\frac{1}{T - T_{env}} \, dT = -k \, dt$$

現在，左邊只剩下跟溫度  $T$  有關的項，右邊只剩下跟時間  $t$  有關的項了。

#### **步驟 2：兩邊同時取不定積分**
既然兩邊的變數已經獨立，我們就可以對等式兩邊分別進行積分：

$$\int \frac{1}{T - T_{env}} \, dT = \int -k \, dt$$

根據微積分的基本公式， $\int \frac{1}{x} dx = \ln|x| + C$ 。因此左邊積分後會產生自然對數（ $\ln$ ）；而右邊是對常數 $-k$ 積分，會補上變數 $t$：

$$\ln|T - T_{env}| = -kt + C$$

(註：雖然兩邊積分都會產生常數，但我們可以將它們合併成一個單一的積分常數 $C$)

#### **步驟 3：利用指數函數解除對數**
我們的目標是求出 $T$ 也就是 $T(t)$，但現在 $T$ 被困在自然對數 $\ln$ 裡面。為了解除 $\ln$，我們將等式兩邊同時放上自然底數 $e$ 的次方（因為 $e^{\ln(x)} = x$）：

$$e^{\ln|T - T_{env}|} = e^{-kt + C}$$

$$|T - T_{env}| = e^{-kt} \cdot e^C$$

#### **步驟 4：處理常數與絕對值，得到「通解」**
- **處理常數**：因為 $e$ 是常數， $C$ 也是常數，所以 $e^C$ 仍然是一個常數。為了版面簡潔，我們令一個新的常數 $C_1 = e^C$。
- **處理絕對值**：在 CPU 散熱的物理情境中，CPU 的溫度 $T$ 必定高於環境溫度 $T_{env}$（即 $T > T_{env}$），這代表 $T - T_{env}$ 永遠是正數。因此，絕對值符號可以直接拿掉。

整理後，我們把 $T_{env}$ 移到右邊，得到溫度的**通解 (General Solution)**：

$$T - T_{env} = C_1 e^{-kt}$$

$$T(t) = T_{env} + C_1 e^{-kt}$$

#### **步驟 5：代入初始條件求解常數 $C_1$**
通解中有一個未知的常數 $C_1$。為了找出它，我們必須代入「初始條件」。
假設在開始測量散熱的瞬間（時間 $t = 0$ 時），CPU 的初始高溫為 $T_0$。將 $t = 0$ 與 $T = T_0$ 代入方程式：

$$T(0) = T_{env} + C_1 e^{-k \cdot 0}$$

因為任何數的 0 次方都是 1（即 $e^0 = 1$），所以方程式簡化為：

$$T_0 = T_{env} + C_1$$

將其移項，解出 $C_1$：

$$C_1 = T_0 - T_{env}$$

*(這在物理上代表了初始狀態下的「溫差」)*

---

### 3. 最終解答（特解）

將求得的 $C_1$ 代回我們在步驟 4 得到的通解中，就得到了最終可以用來預測 CPU 溫度的數學模型：

$$T(t) = T_{env} + (T_0 - T_{env})e^{-kt}$$
